What I do

Development

I do everything from web design to app development. I create custom command line tools, and apps with sleek and modern UI's, both of which often utilize backend SQL and NoSql databases. Whatever your need I'm certain I can find a solution using one of the several programming languages with which I am familiar. These include Python, C#, JavaScript, SQL(including Postgres, MySQL, SQLite), Bash, as well as markup languages like HTML and YAML, and of course style sheet languages like CSS. I will always strive to find the most appropriate languages and tools for a given project and if that means learning a new language I am not only capable but excited to do so in a timely manner.

Linux System Administration

I absolutley love Linux! I spent over 100 hours just taking video courses when I first met Tux. I've spent hundreds more working on projects like setting up network fileshares between various devices and virtual machines, learning cyber security practices on sites like hackthebox.com, overthewire.org, hackthissite.org and my favorite tryhackme.com. I started my adventures with python on a Linux distro and my first few projects were ethical hacking tools like webcrawlers, vulnerability scanners, backdoor and listener etc. You can find all of the code for these projects on my github: TheCliCommander
